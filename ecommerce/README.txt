	Dulce Argentina
	===============


    1)	Descripción general del sistema o proyecto
    	------------------------------------------
    		Página Web que es un ecommerce, en donde se venden productos regionales argentinos para el Mundo.

    
    2)	Estado del proyecto
    	-------------------
    		El Proyecto todavía está en desarrollo, esta previsto continuar con funcionalidades de usuarios y mejoras de vistas en los HTML. También a futuro se le piensa agregar más productos o combos de varios productos como para seguir agregandoles funcionalidad al Proyecto.
    		
    		    
    3)	Requisitos
    	----------
    		El Proyecto puede visualizarse en cualquier computadora que tengo algun Navegador de Internet y conexión a Internet.
    
    
    4)	Instalación
    	-----------
    	No requiere, solo ingresar a la URL indicada.
    	
    	
    5)	Tecnologías utilizadas
    	----------------------
    	El Proyecto fue desarrollado en Django que es un Framework de Python, y HTML para la visualización.
